# Indian Bikes Sales Analysis Dashboard

## Overview

This project is an interactive Microsoft Excel dashboard created to analyze Indian bike sales data.

## Features

- KPI Cards
- Brand-wise Analysis
- State-wise Analysis
- Fuel Type Distribution
- Original vs Resale Price Comparison
- Top Selling Models
- Interactive Slicers

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
